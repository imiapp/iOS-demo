# iOS-demo
iOS demo using IMI SDK
### demo中设置了测试环境的api和正式环境的api，开发人员可以自由切换正式环境和测试环境。
### 注意点：demo中的测试环境和正式环境分别需要配合IMI的测试版本和正式版本就行调试。
    demo中的调试环境默认为正式版本。


#### 使用说明

- 请扫描二维码并下载应用商城对应的正式版本进行测试和使用
